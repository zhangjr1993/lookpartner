# 启动图片资源目录

## 说明
此目录用于存放应用的启动页图片资源。

## 当前图片
- `bg_launch.png` - 应用启动页主图片
  - 描述：带有"纯悦"字样的黄色圆角方形图标
  - 背景：从浅奶油色到橙黄色的渐变
  - 用途：应用启动页显示

## 注意事项
- 图片格式：PNG（推荐）或 WebP
- 分辨率：建议使用高分辨率图片以确保在不同设备上的显示效果
- 文件大小：建议控制在合理范围内以优化应用启动速度

## 使用方法
在代码中通过以下路径引用：
```dart
'assets/images/launchAssets/bg_launch.png'
```
